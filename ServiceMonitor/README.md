* INSTRUCTIONS: 

Command line arguments: 
* -s | --speed:
	* Set the program speed. 1 by default. 2 = twice as fast, 3 = thrice as fast, etc.
	* Use decimals to go slower. 0.5 = half the speed, 0.25 = quarter the speed, etc.
	* 0 and negative values not accepted.
